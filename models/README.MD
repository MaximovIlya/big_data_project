# models/

This directory contains trained Spark MLlib `PipelineModel` objects saved after Stage III.

## Contents

| Directory | Model | Algorithm |
|---|---|---|
| `random_forest/` | Random Forest Classifier | `pyspark.ml.classification.RandomForestClassifier` |
| `linear_svc/` | Linear SVC (One-vs-Rest) | `pyspark.ml.classification.LinearSVC` + `OneVsRest` |
| `naive_bayes/` | Naive Bayes (Multinomial) | `pyspark.ml.classification.NaiveBayes` |

Models are generated by `bash scripts/stage3.sh` and populated during pipeline execution.

## Model Details

### Random Forest
- **Task:** Multi-class classification (11 classes)
- **Hyperparameter tuning:** 27 combinations (numTrees × maxDepth × maxBins), 5-fold CV
- **Best params:** `numTrees=200`, `maxDepth=15`
- **Accuracy:** 0.3644 | **Weighted F1:** 0.2559

### Linear SVC (One-vs-Rest)
- **Task:** Multi-class via OvR decomposition (11 binary classifiers)
- **Hyperparameter tuning:** 27 combinations (regParam × maxIter × tol), 5-fold CV
- **Best params:** `regParam=0.1`, `maxIter=50`
- **Accuracy:** 0.2701 | **Weighted F1:** 0.2232

### Naive Bayes
- **Task:** Multi-class classification (multinomial variant)
- **Hyperparameter tuning:** 5 smoothing values, 5-fold CV
- **Best params:** `smoothing=1.5`
- **Accuracy:** 0.3276 | **Weighted F1:** 0.2248

## Loading a Saved Model

```python
from pyspark.ml import PipelineModel

model = PipelineModel.load("models/random_forest")
predictions = model.transform(test_df)
```

## Notes

- Models are excluded from git (listed in `.gitignore`) — run `bash main.sh` to generate them
- Each model directory is a Spark `PipelineModel` that includes all preprocessing stages (StringIndexer, OneHotEncoder, VectorAssembler, Scaler, Classifier, IndexToString)
- Models are saved with `.write().overwrite().save()` so re-runs replace existing files
